To set up the backend server and database:
==========================================

##1. Install dependencies: 
--------------------------
    ```bash
        npm install
    ```
##2. To setup database container with the sql file specified:
-------------------------------------------------------------
    ```bash
        npm run createdb
    ```
##3. After you make some changes to the database, you want to export it, run:
-----------------------------------------------------------------------------
    ```bash
        npm run extractdb
    ```
